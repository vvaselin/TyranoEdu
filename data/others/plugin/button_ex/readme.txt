【buttonタグの機能補助するプラグイン】

■使い方
このテキストが入っているフォルダごと「data/others/plugin」フォルダに置きます。
その後、first.ksとかに以下のように記述してください。

[plugin name=button_ex]
指定可能属性：なし

記述した時点から後述のタグを使用可能になります。
また、セーブデータロード時の処理のため、make.ksに以下のように記述してください。

[button_ex_restore]


■できること
・ボタンにマウスオーバー時、フェードしながらボタン画像切り替え
・ボタンにマウスオーバー時、ツールチップとして任意の画像を表示
・ツールチップ画像をアニメーションしながら表示／非表示
・任意のJS式がtrueのとき、ボタンを無効化
・ボタン画像の透明部分の操作を無効に


■タグと指定可能属性
button_ex   ：ボタン機能拡張
属性：
name        ：buttonタグと同じnameを指定します。
enter_fade  ：ボタン画像を切り替える時間を数値で指定します。単位はミリ秒です。
tip         ：マウスオーバー時に表示するツールチップ画像を指定します。画像はimageフォルダに配置してください。
tip_pos     ：ツールチップ画像をボタンから見て上下左右のどこに表示するか指定します。"top","bottom","left","right","static"のいずれかを指定してください。
tip_margin  ：ツールチップ画像とボタンの間隔を数値で指定します。
tip_fade    ：ツールチップ画像を表示／非表示する時間を数値で指定します。単位はミリ秒です。
tip_method  ：ツールチップ画像を表示／非表示時のアニメーションを指定します。bgタグのmethodと同じものが指定できます。初期値は"fadeIn"です。
tip_x       ：tip_posがstaticのとき、ツールチップ画像を表示する横位置を数値で指定します。単位はピクセルです。
tip_y       ：tip_posがstaticのとき、ツールチップ画像を表示する縦位置を数値で指定します。単位はピクセルです。
disable     ：指定したJS式がtrueのとき、ボタンを無効にします。
disableimg  ：ボタン無効化時の画像を指定します。
alpha       ：指定した値未満の透明度部分を操作無効にします。「0」で透明部分まで含めたボタン画像全体が操作有効になります。

button_ex_restore：make.ks記述用（必ず「make.ks」にこのタグを記述してください）
属性：
なし


■使い方
[button name="button_test" graphic="test.png" target="*label" x=100 y=100]
[button_ex name="button_test" enter_fade=100 tip="button_test_tip.png" tip_pos="top"]

必ず[button]タグとセットで使います。
親となるボタンがない場合、エラーとなります。


■注意事項
