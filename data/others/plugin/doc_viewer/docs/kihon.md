## プログラミングって何だろう？

みなさんは料理のレシピを見たことがありますか？

> 1. 玉ねぎをみじん切りにする
> 2. フライパンに油を引いて火をつける
> 3. 玉ねぎを炒める
> 4. ...

レシピは「料理をどういう順番で、どうやって作るか」を人間に伝えるための指示書です。

**プログラムも、これと同じです。**
ただし、指示を届ける相手が「人間」ではなく「コンピュータ」になります。

コンピュータはとても忠実です。書かれた指示を、書かれた通りにそのまま実行します。
だからこそ、**正確に指示を書く**ことがとても大切になります。

---

## プログラミング言語って何？

コンピュータに指示を伝えるには、コンピュータが理解できる言葉を使わなければなりません。
その言葉のことを **プログラミング言語** といいます。

プログラミング言語にはたくさんの種類があります。

| 言語 | 主な用途 |
|------|---------|
| Python | データ分析・AI・スクリプト |
| JavaScript | Webサイトの動き |
| Java | スマホアプリ・業務システム |
| **C++** | ゲーム・組み込み・高速な処理 |

それぞれ得意なことが違います。このサイトでは **C++** を学んでいきます。

---

## C++ってどんな言語？

C++は1983年に作られた、歴史のあるプログラミング言語です。

**C++の特徴：**

- ⚡ **速い** — コンピュータが直接理解できる形に変換されるので、処理がとても高速です
- 🎮 **幅広く使われている** — ゲームエンジン（UnrealEngineなど）、ブラウザ、OSなど、身近なソフトウェアの多くがC++で作られています
- 📚 **しっかり学べる** — プログラミングの基礎的な考え方をしっかり身につけられます

少し難しい部分もありますが、一緒に丁寧に進んでいきましょう！

## プログラムが動くまでの流れ

C++のプログラムは、次のような流れで動きます。

```
① ソースコード（.cpp）を書く
        ↓
② コンパイル（コンピュータが読める形に変換する）
        ↓
③ 実行ファイルができる
        ↓
④ 実行！
```

**ソースコード** とは、私たちが書くプログラムの文章のことです。
**コンパイル** とは、そのソースコードをコンピュータが直接実行できる形に変換する作業のことです。

> 💡 「コンパイル」という作業が必要なのがC++の特徴です。Pythonなどはコンパイル不要で、すぐに実行できます。

## はじめてのC++プログラムを見てみよう

こちらが、世界中のプログラマーが最初に書くといわれる定番プログラムです。

```cpp
#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```

↓ こう書くこともできます。

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}
```

実行すると、画面にこう表示されます：

```
Hello, World!
```

2つのコードは全く同じ動きをします。
`using namespace std;` を1行追加することで、`std::cout` を `cout` と短く書けるようになります。
この教材では、読みやすさのために **2つ目の書き方** を使っていきます。

今はまだ意味がわからなくても大丈夫です。
第1章から順番に学んでいくうちに、自然と理解できるようになります。

---

## まとめ

- プログラムとは、コンピュータへの**指示書**のこと
- 指示を書くための言葉を**プログラミング言語**という
- C++は**速くて幅広く使われる**プログラミング言語
- C++ではソースコードを**コンパイル**してから実行する
