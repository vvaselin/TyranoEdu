## `#include` って何をしているの？

プログラムの先頭にいつも書いてきた `#include`。
これは「このファイルの機能を使います」という宣言です。

```cpp
#include <iostream>   // cout, cin など
#include <vector>     // vector
#include <string>     // string
```

C++の標準ライブラリは機能ごとにファイルが分かれており、
使いたい機能に対応するファイルを `#include` で読み込む必要があります。

## よく使う標準ライブラリ一覧

### 入出力

| ヘッダ | 主な機能 |
|--------|---------|
| `<iostream>` | `cin` / `cout` / `cerr` |
| `<fstream>` | ファイルの読み書き |
| `<sstream>` | 文字列ストリーム |
| `<iomanip>` | 出力フォーマット（小数点桁数など） |

### 文字列・文字

| ヘッダ | 主な機能 |
|--------|---------|
| `<string>` | `string` クラス |
| `<cctype>` | `isdigit()` / `toupper()` など文字判定・変換 |
| `<regex>` | 正規表現 |

### コンテナ

| ヘッダ | 主な機能 |
|--------|---------|
| `<vector>` | `vector` |
| `<array>` | 固定長配列 `array` |
| `<map>` | `map` / `multimap` |
| `<unordered_map>` | `unordered_map` |
| `<set>` | `set` / `multiset` |
| `<unordered_set>` | `unordered_set` |
| `<queue>` | `queue` / `priority_queue` |
| `<stack>` | `stack` |
| `<deque>` | 両端キュー `deque` |
| `<list>` | 双方向リスト `list` |
| `<tuple>` | `tuple` |
| `<utility>` | `pair` / `move` など |

### アルゴリズム・数値

| ヘッダ | 主な機能 |
|--------|---------|
| `<algorithm>` | `sort()` / `find()` / `min()` / `max()` など |
| `<numeric>` | `accumulate()` / `iota()` など |
| `<cmath>` | `sqrt()` / `pow()` / `abs()` など数学関数 |
| `<cstdlib>` | `rand()` / `abs()` など |
| `<climits>` | `INT_MAX` / `INT_MIN` など整数の限界値 |
| `<cfloat>` | `DBL_MAX` など浮動小数点の限界値 |

### 時間・乱数

| ヘッダ | 主な機能 |
|--------|---------|
| `<random>` | 乱数生成（現代的な方法） |
| `<chrono>` | 時間計測 |

## `#include <bits/stdc++.h>` という書き方

競技プログラミングでよく見かける書き方です。

```cpp
#include <bits/stdc++.h>
```

これ1行で、ほぼすべての標準ライブラリをまとめて読み込めます。
競プロのように「手早く書くことを優先する」場面では便利ですが、
コンパイルが遅くなるため**実務では使わない**のが一般的です。

また、GCC（Linux/Mac）では使えますが、**Visual StudioなどのWindowsの環境では使えない**場合があります。

---

## よく使う数学関数（`<cmath>`）

```cpp
#include <cmath>

sqrt(9.0)      // → 3.0（平方根）
pow(2.0, 10)   // → 1024.0（べき乗）
abs(-5)        // → 5（絶対値）
floor(3.7)     // → 3.0（切り捨て）
ceil(3.2)      // → 4.0（切り上げ）
round(3.5)     // → 4.0（四捨五入）
```

---

## 出力のフォーマット（`<iomanip>`）

```cpp
#include <iomanip>

double pi = 3.14159265;

// 小数点以下3桁で表示
cout << fixed << setprecision(3) << pi << endl;   // → 3.142

// 幅を10文字分確保して右揃え
cout << setw(10) << 42 << endl;   // →         42
```

---

## まとめ

- `#include` で使いたいライブラリのヘッダを読み込む
- 機能ごとにヘッダが分かれている（`<vector>`, `<algorithm>` など）
- `<bits/stdc++.h>` は競プロ向けの一括読み込み（環境依存あり）
- `<cmath>` で数学関数、`<iomanip>` で出力フォーマットが使える
