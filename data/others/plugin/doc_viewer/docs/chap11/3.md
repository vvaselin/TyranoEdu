## `<algorithm>` ライブラリ

C++には、よく使うアルゴリズムが `<algorithm>` にまとまっています。
自分で実装しなくても、1行で使えるものが多いです。

```cpp
#include <algorithm>
```

---

## ソート（sort）

`sort()` で配列や vector を並べ替えます。

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    vector<int> v = {5, 2, 8, 1, 9, 3};

    sort(v.begin(), v.end());   // 昇順ソート

    for (int x : v) {
        cout << x << " ";
    }
    // → 1 2 3 5 8 9

    return 0;
}
```

### 降順ソート

```cpp
sort(v.begin(), v.end(), greater<int>());   // 降順
```

### 独自の比較関数

```cpp
// 絶対値が小さい順にソート
sort(v.begin(), v.end(), [](int a, int b) {
    return abs(a) < abs(b);
});
```

`[](int a, int b){ return ...; }` はラムダ式といい、「その場で作る小さな関数」です。

---

## 最大・最小（max / min）

```cpp
cout << max(3, 7) << endl;   // → 7
cout << min(3, 7) << endl;   // → 3
```

3つ以上の比較は `max({a, b, c})` のように書けます。

```cpp
cout << max({1, 5, 3, 2}) << endl;   // → 5
cout << min({1, 5, 3, 2}) << endl;   // → 1
```

### vector内の最大・最小

```cpp
vector<int> v = {5, 2, 8, 1, 9};
cout << *max_element(v.begin(), v.end()) << endl;   // → 9
cout << *min_element(v.begin(), v.end()) << endl;   // → 1
```

## 合計（accumulate）

```cpp
#include <numeric>

vector<int> v = {1, 2, 3, 4, 5};
int sum = accumulate(v.begin(), v.end(), 0);   // 初期値 0 から足す
cout << sum << endl;   // → 15
```

---

## 要素の検索（find）

```cpp
vector<int> v = {10, 20, 30, 40, 50};
auto it = find(v.begin(), v.end(), 30);

if (it != v.end()) {
    cout << "見つかりました: " << *it << endl;   // → 30
    cout << "位置: " << it - v.begin() << endl;  // → 2
} else {
    cout << "見つかりませんでした" << endl;
}
```

---

## 条件を満たす要素の数（count / count_if）

```cpp
vector<int> v = {1, 2, 3, 2, 5, 2};

// 値が 2 の要素の数
cout << count(v.begin(), v.end(), 2) << endl;   // → 3

// 3以上の要素の数
cout << count_if(v.begin(), v.end(), [](int x) {
    return x >= 3;
}) << endl;   // → 2
```

---

## 逆順（reverse）

```cpp
vector<int> v = {1, 2, 3, 4, 5};
reverse(v.begin(), v.end());

for (int x : v) cout << x << " ";
// → 5 4 3 2 1
```

---

## 重複の削除（unique）

ソート済みの配列から、連続する重複要素を取り除きます。

```cpp
vector<int> v = {1, 1, 2, 3, 3, 3, 4};
auto newEnd = unique(v.begin(), v.end());
v.erase(newEnd, v.end());

for (int x : v) cout << x << " ";
// → 1 2 3 4
```

ソートしてから `unique` をかけることで重複排除ができます。

```cpp
sort(v.begin(), v.end());
v.erase(unique(v.begin(), v.end()), v.end());
```

---

## 二分探索（binary_search / lower_bound）

**ソート済みの配列**に対して、高速に要素を探せます。

```cpp
vector<int> v = {1, 3, 5, 7, 9};   // ソート済み

// 存在するか確認
cout << binary_search(v.begin(), v.end(), 5) << endl;   // → 1（true）
cout << binary_search(v.begin(), v.end(), 4) << endl;   // → 0（false）

// 5 以上の最初の位置
auto it = lower_bound(v.begin(), v.end(), 5);
cout << *it << endl;              // → 5
cout << it - v.begin() << endl;   // → 2（インデックス）
```

---

## fill — 一括で同じ値を入れる

```cpp
vector<int> v(5);
fill(v.begin(), v.end(), 0);   // 全要素を 0 に
```

## アルゴリズム一覧（よく使うもの）

| 関数 | 概要 | ヘッダ |
|------|------|--------|
| `sort()` | ソート | `<algorithm>` |
| `max()` / `min()` | 最大・最小 | `<algorithm>` |
| `max_element()` / `min_element()` | 配列内の最大・最小 | `<algorithm>` |
| `find()` | 線形探索 | `<algorithm>` |
| `count()` / `count_if()` | 要素のカウント | `<algorithm>` |
| `reverse()` | 逆順 | `<algorithm>` |
| `unique()` | 重複除去（要ソート） | `<algorithm>` |
| `binary_search()` | 二分探索（要ソート） | `<algorithm>` |
| `lower_bound()` | 境界値検索 | `<algorithm>` |
| `fill()` | 一括代入 | `<algorithm>` |
| `accumulate()` | 合計 | `<numeric>` |

---

## まとめ

- `<algorithm>` にはソート・検索・集計など多くの関数が揃っている
- `sort()` は `v.begin()` と `v.end()` を渡して使う
- ラムダ式 `[](型 a, 型 b){ return ...; }` で比較条件を自由に指定できる
- 二分探索系（`binary_search` / `lower_bound`）は**ソート済みが前提**
