## 文字列をもっと使いこなそう

第2章で `string` 型を紹介しました。
このページでは、文字列に対してできる操作を詳しく学びます。

```cpp
#include <iostream>
#include <string>
using namespace std;
```

`string` を使うときは `#include <string>` を忘れずに。

---

## 文字列の長さを取得する

`.size()` または `.length()` で文字列の長さを取得できます。

```cpp
string name = "Hello";
cout << name.size() << endl;     // → 5
cout << name.length() << endl;   // → 5（同じ結果）
```

---

## 文字列を連結する

`+` 演算子で文字列をつなげられます。

```cpp
string first = "Hello";
string second = ", World!";
string message = first + second;

cout << message << endl;   // → Hello, World!
```

`+=` で後ろに追加することもできます。

```cpp
string text = "Hello";
text += ", World!";
cout << text << endl;   // → Hello, World!
```

---

## 1文字ずつ取り出す

`[インデックス]` で特定の位置の文字を取り出せます。
配列と同じように、インデックスは **0から始まります**。

```cpp
string word = "Hello";
cout << word[0] << endl;   // → H
cout << word[1] << endl;   // → e
cout << word[4] << endl;   // → o
```

for文と組み合わせて1文字ずつ処理することもできます。

```cpp
string word = "Hello";

for (int i = 0; i < word.size(); i++) {
    cout << word[i] << endl;
}
```

実行結果：
```
H
e
l
l
o
```

---

## 文字列の一部を取り出す（substr）

`.substr(開始位置, 文字数)` で文字列の一部を切り出せます。

```cpp
string text = "Hello, World!";

cout << text.substr(0, 5) << endl;    // → Hello
cout << text.substr(7, 5) << endl;    // → World
cout << text.substr(7) << endl;       // → World!（最後まで）
```

第2引数を省略すると、開始位置から最後まで取り出します。

---

## 文字列を検索する（find）

`.find(検索文字列)` で、特定の文字列が何文字目にあるかを調べられます。

```cpp
string text = "Hello, World!";
int pos = text.find("World");

cout << pos << endl;   // → 7
```

見つからなかった場合は `string::npos` という特別な値が返ります。

```cpp
if (text.find("C++") == string::npos) {
    cout << "見つかりませんでした" << endl;
}
```

---

## 文字列を比較する

`==` で2つの文字列が同じかどうかを比較できます。

```cpp
string a = "Hello";
string b = "Hello";
string c = "World";

if (a == b) {
    cout << "aとbは同じ" << endl;   // こちらが表示される
}

if (a != c) {
    cout << "aとcは違う" << endl;   // こちらも表示される
}
```

---

## 数値と文字列を変換する

### 数値 → 文字列：`to_string()`

```cpp
int num = 42;
string s = to_string(num);
cout << s << endl;         // → "42"
cout << s + "点" << endl;  // → "42点"
```

### 文字列 → 整数：`stoi()`

```cpp
string s = "123";
int num = stoi(s);
cout << num + 1 << endl;   // → 124
```

> 💡 `stoi` は **s**tring **to** **i**nteger の略です。

---

## まとめ

| 操作 | 書き方 | 例 |
|------|--------|----|
| 長さを取得 | `.size()` | `name.size()` |
| 連結 | `+` または `+=` | `a + b` |
| 1文字取り出し | `[インデックス]` | `word[0]` |
| 一部を切り出し | `.substr(開始, 文字数)` | `text.substr(0, 5)` |
| 検索 | `.find(文字列)` | `text.find("Hello")` |
| 比較 | `==` / `!=` | `a == b` |
| 数値→文字列 | `to_string()` | `to_string(42)` |
| 文字列→整数 | `stoi()` | `stoi("42")` |
