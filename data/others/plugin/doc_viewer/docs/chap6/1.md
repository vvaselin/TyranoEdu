## 複数の値をまとめて管理しよう

5人のテストの点数を管理したいとします。
変数を1つずつ用意すると、こうなります。

```cpp
int score1 = 80;
int score2 = 65;
int score3 = 90;
int score4 = 72;
int score5 = 88;
```

これでも動きますが、100人分になったら大変です。
**配列** を使うと、同じ型の値をまとめて管理できます。

```cpp
int scores[5] = {80, 65, 90, 72, 88};
```

たった1行でまとめられました！

---

## 配列の宣言

```cpp
型 配列名[要素数];
```

```cpp
int scores[5];      // int型の値を5つ入れられる配列
double heights[3];  // double型の値を3つ入れられる配列
```

宣言と同時に値を入れることもできます。

```cpp
int scores[5] = {80, 65, 90, 72, 88};
```

---

## 配列のイメージ

配列は、**番号のついた箱が並んだもの**です。

```
      [0]  [1]  [2]  [3]  [4]
     ┌────┬────┬────┬────┬────┐
     │ 80 │ 65 │ 90 │ 72 │ 88 │
     └────┴────┴────┴────┴────┘
scores
```

番号のことを **インデックス（添字）** といいます。
インデックスは **0から始まる**ことに注意してください。

---

## 配列の要素にアクセスする

`配列名[インデックス]` で特定の要素を読み書きできます。

```cpp
#include <iostream>
using namespace std;

int main() {
    int scores[5] = {80, 65, 90, 72, 88};

    cout << scores[0] << endl;   // → 80（1番目）
    cout << scores[2] << endl;   // → 90（3番目）
    cout << scores[4] << endl;   // → 88（5番目）

    scores[1] = 70;   // 2番目の値を変更
    cout << scores[1] << endl;   // → 70

    return 0;
}
```

---

## for文と組み合わせる

配列とfor文は相性抜群です。
全要素に同じ処理をするときに、とても短く書けます。

```cpp
#include <iostream>
using namespace std;

int main() {
    int scores[5] = {80, 65, 90, 72, 88};

    // 全要素を表示
    for (int i = 0; i < 5; i++) {
        cout << "scores[" << i << "] = " << scores[i] << endl;
    }

    return 0;
}
```

実行結果：
```
scores[0] = 80
scores[1] = 65
scores[2] = 90
scores[3] = 72
scores[4] = 88
```

---

## 合計・平均を求める

```cpp
#include <iostream>
using namespace std;

int main() {
    int scores[5] = {80, 65, 90, 72, 88};
    int sum = 0;

    for (int i = 0; i < 5; i++) {
        sum += scores[i];
    }

    double average = (double)sum / 5;

    cout << "合計: " << sum << endl;
    cout << "平均: " << average << endl;

    return 0;
}
```

実行結果：
```
合計: 395
平均: 79
```

## 範囲forループ（for-each）

C++11以降では、配列の全要素を簡単に取り出せる書き方があります。

```cpp
for (型 変数名 : 配列名) {
    // 処理
}
```

```cpp
int scores[5] = {80, 65, 90, 72, 88};

for (int s : scores) {
    cout << s << endl;
}
```

インデックスが不要で全要素を順番に処理するだけなら、こちらの方がシンプルです。

---

## よくあるミス

### ミス1: インデックスが範囲外

```cpp
int scores[5] = {80, 65, 90, 72, 88};
cout << scores[5] << endl;   // ← インデックスは 0〜4 まで！
```

要素数が5の配列のインデックスは `0` から `4` です。
`scores[5]` は存在しないため、予期しない動作になります（エラーにならないこともあり、その分危険です）。

### ミス2: 0始まりを忘れる

```cpp
// 1番目の要素を取得したい
cout << scores[1] << endl;   // ← 実は2番目！
cout << scores[0] << endl;   // ← これが1番目
```

---

## まとめ

- 配列は**同じ型の値を複数まとめて管理**できる
- インデックスは **0から始まる**
- `for`文と組み合わせると全要素を効率よく処理できる
- 範囲forループ（`for (型 変数 : 配列)`）で全要素を簡単に取り出せる
