# C++入門教材へようこそ！

この資料は、**プログラミングをはじめて学ぶ方**のためのC++入門教材です。

「プログラムって何？」というところから、やさしく丁寧に解説していきます。
難しい言葉はできるだけ使わず、たくさんの例を交えながら進めていきますので、安心して読み進めてください。

---

## この教材の使い方

1. まず「**プログラムの基本**」を読んで、プログラミングの雰囲気をつかみましょう
2. あとは**第1章から順番に**読み進めていくのがおすすめです
3. サンプルコードは**実際に手を動かして試してみる**と理解が深まります

> 💡 わからないところがあっても、とりあえず先に進んでみてください。読み返すうちに「あ、そういうことか！」となることがよくあります。
