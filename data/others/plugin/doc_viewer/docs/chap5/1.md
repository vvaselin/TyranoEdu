## 同じ処理を繰り返そう

「1から10まで数字を表示したい」とき、こんなコードを書くのは大変です。

```cpp
cout << 1 << endl;
cout << 2 << endl;
cout << 3 << endl;
// ...10行続く
```

こういった**繰り返しの処理**を短くスッキリ書けるのが **繰り返し文** です。
この章では、回数が決まった繰り返しに使う **`for`文** を学びます。

---

## for文の基本の形

```cpp
for (初期化; 条件; 更新) {
    // 繰り返したい処理
}
```

3つのパーツがあります。

| パーツ | タイミング | 例 |
|--------|-----------|-----|
| 初期化 | 最初に1回だけ実行 | `int i = 0` |
| 条件 | 繰り返すたびに確認 | `i < 5` |
| 更新 | 1回の処理が終わるたびに実行 | `i++` |

条件が `true` の間、`{}` の中を繰り返し実行します。

---

## 基本的な例

```cpp
#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 5; i++) {
        cout << i << endl;
    }
    return 0;
}
```

実行結果：
```
0
1
2
3
4
```

実行の流れを追ってみましょう。

```
① int i = 0  （初期化）
② i < 5 → true  → cout << 0
③ i++  → i = 1
④ i < 5 → true  → cout << 1
⑤ i++  → i = 2
  ...
⑥ i < 5 → false → ループ終了
```

---

## 1から始めたいとき

```cpp
for (int i = 1; i <= 5; i++) {
    cout << i << endl;
}
```

実行結果：
```
1
2
3
4
5
```

`i = 0` か `i = 1` かで条件も変わります。
- `i = 0` スタートなら条件は `i < 5`
- `i = 1` スタートなら条件は `i <= 5`

どちらでも5回繰り返すことができます。

---

## ループ変数を計算に使う

`i` の値は `{}` の中で自由に使えます。

```cpp
#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 5; i++) {
        cout << i << "の2乗は" << i * i << endl;
    }
    return 0;
}
```

実行結果：
```
1の2乗は1
2の2乗は4
3の2乗は9
4の2乗は16
5の2乗は25
```

---

## 合計を計算する

ループと変数を組み合わせると、合計の計算が簡単に書けます。

```cpp
#include <iostream>
using namespace std;

int main() {
    int sum = 0;

    for (int i = 1; i <= 100; i++) {
        sum += i;   // sum = sum + i
    }

    cout << "1から100の合計: " << sum << endl;
    return 0;
}
```

実行結果：
```
1から100の合計: 5050
```

`sum += i` で、ループのたびに `i` の値を `sum` に積み上げていきます。

---

## 逆順に繰り返す

減らしながら繰り返すこともできます。

```cpp
for (int i = 5; i >= 1; i--) {
    cout << i << endl;
}
```

実行結果：
```
5
4
3
2
1
```

`i--` で1ずつ減らしています。

---

## 2ずつ増やす

```cpp
for (int i = 0; i <= 10; i += 2) {
    cout << i << endl;
}
```

実行結果：
```
0
2
4
6
8
10
```

`i++` の代わりに `i += 2` にすると2ずつ増やせます。

---

## よくあるミス

### ミス1: 無限ループ

```cpp
for (int i = 0; i < 5; i--) {   // ← i-- で減り続ける！
    cout << i << endl;
}
```

条件が永遠に `true` のままになると、プログラムが止まらなくなります。
これを **無限ループ** といいます。
更新式の向きが条件と合っているか確認しましょう。

### ミス2: セミコロンをつけてしまう

```cpp
for (int i = 0; i < 5; i++);   // ← ; がついている！
{
    cout << i << endl;   // ← ループの外になってしまう
}
```

`for (...)` の直後に `;` をつけると、`{}` の中がループされません。

---

## まとめ

- `for`文は**回数が決まった繰り返し**に使う
- `for (初期化; 条件; 更新)` の3つのパーツで制御する
- ループ変数は `{}` の中で使える
- 更新式を変えることで、逆順・2ずつなど柔軟に繰り返せる
