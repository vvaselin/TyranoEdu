## while文とは

`for`文は「何回繰り返すか」があらかじめ決まっているときに使いました。
**`while`文** は「条件が成り立つ間、繰り返す」という書き方です。

回数ではなく、**状態によって繰り返しを続けるかどうか決める**ときに向いています。

---

## while文の基本の形

```cpp
while (条件) {
    // 条件が true の間、繰り返す処理
}
```

`for`文と違い、初期化・更新は `while` の外に自分で書きます。

---

## 基本的な例

```cpp
#include <iostream>
using namespace std;

int main() {
    int i = 0;

    while (i < 5) {
        cout << i << endl;
        i++;
    }

    return 0;
}
```

実行結果：
```
0
1
2
3
4
```

`for`文で書くと次のようになります。どちらも同じ動きです。

```cpp
for (int i = 0; i < 5; i++) {
    cout << i << endl;
}
```

回数が決まっているときは `for`、そうでないときは `while` が自然です。

---

## while文が向いている場面

### ユーザーの入力によって繰り返す

「`0` が入力されるまで数字を受け取り続ける」といった処理は、
何回繰り返すか事前にわからないので `while` が向いています。

```cpp
#include <iostream>
using namespace std;

int main() {
    int num = -1;

    while (num != 0) {
        cout << "数字を入力してください（0で終了）: ";
        cin >> num;
        cout << "入力された値: " << num << endl;
    }

    cout << "終了しました。" << endl;
    return 0;
}
```

実行例：
```
数字を入力してください（0で終了）: 5
入力された値: 5
数字を入力してください（0で終了）: 12
入力された値: 12
数字を入力してください（0で終了）: 0
入力された値: 0
終了しました。
```

---

## break：ループを途中で抜ける

`break` を使うと、ループを途中で抜けることができます。

```cpp
#include <iostream>
using namespace std;

int main() {
    int i = 0;

    while (true) {   // 条件が常に true → 無限ループ
        cout << i << endl;
        i++;
        if (i >= 3) {
            break;   // ← ループを抜ける
        }
    }

    cout << "ループ終了" << endl;
    return 0;
}
```

実行結果：
```
0
1
2
ループ終了
```

`while (true)` は「永遠に繰り返す」という書き方ですが、
`break` を使って適切なタイミングで抜けることでコントロールできます。

---

## continue：今回だけスキップする

`continue` を使うと、その回の処理だけスキップして次の繰り返しに進みます。

```cpp
#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 10; i++) {
        if (i % 2 == 0) {
            continue;   // 偶数のときはスキップ
        }
        cout << i << endl;   // 奇数のときだけ表示
    }

    return 0;
}
```

実行結果：
```
1
3
5
7
9
```

> 💡 `break` は「ループから出る」、`continue` は「今回だけ飛ばして続ける」と覚えましょう。

---

## do-while文：必ず1回は実行する

`while`文は条件を最初にチェックするため、最初から条件が `false` だと1回も実行されません。

```cpp
int i = 10;
while (i < 5) {
    cout << i << endl;   // 一度も実行されない
}
```

**必ず1回は実行したい**ときは `do-while` を使います。

```cpp
do {
    // 処理
} while (条件);
```

```cpp
#include <iostream>
using namespace std;

int main() {
    int num;

    do {
        cout << "1以上の数を入力してください: ";
        cin >> num;
    } while (num < 1);   // 1未満なら繰り返す

    cout << "入力された値: " << num << endl;
    return 0;
}
```

実行例：
```
1以上の数を入力してください: -3
1以上の数を入力してください: 0
1以上の数を入力してください: 5
入力された値: 5
```

入力の受け取りのように「まず1回やってみて、ダメなら繰り返す」という場面で役立ちます。

---

## for・while・do-whileの使い分け

| 構文 | 向いている場面 |
|------|---------------|
| `for` | 繰り返す回数が決まっているとき |
| `while` | 条件によって繰り返すかどうか決まるとき |
| `do-while` | 必ず1回は実行して、その後条件を確認するとき |

迷ったら `for` から考えて、合わなければ `while` を使うという判断で大丈夫です。

---

## まとめ

- `while` は**条件が true の間**繰り返す
- 回数が不定のとき・状態によって終わりが変わるときに向いている
- `break` でループを途中で抜けられる
- `continue` でその回だけスキップできる
- `do-while` は必ず1回実行してから条件をチェックする
