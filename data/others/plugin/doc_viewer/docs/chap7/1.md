## 同じ処理をまとめよう

プログラムを書いていると、同じような処理を何度も書くことがあります。

```cpp
// 1回目
cout << "========" << endl;
cout << "  結果  " << endl;
cout << "========" << endl;

// 2回目（また同じことを書く…）
cout << "========" << endl;
cout << "  結果  " << endl;
cout << "========" << endl;
```

これを **関数** にまとめると、1回書くだけで何度でも呼び出せます。

```cpp
void showResult() {
    cout << "========" << endl;
    cout << "  結果  " << endl;
    cout << "========" << endl;
}

// 呼び出すだけ
showResult();
showResult();
```

## 関数の基本の形

```cpp
戻り値の型 関数名(引数の型 引数名, ...) {
    // 処理
    return 戻り値;
}
```

難しそうに見えますが、パーツを一つずつ見ていきましょう。

---

## 何も返さない関数（void）

処理をまとめるだけで、結果を返す必要がない関数には `void` を使います。

```cpp
#include <iostream>
using namespace std;

void greet() {
    cout << "こんにちは！" << endl;
}

int main() {
    greet();   // 関数を呼び出す
    greet();
    greet();
    return 0;
}
```

実行結果：
```
こんにちは！
こんにちは！
こんにちは！
```

`void` は「何も返さない」という意味です。
`return` は省略できます（または `return;` と書きます）。

---

## 引数：関数に値を渡す

関数に値を渡して、その値を使った処理をさせることができます。
渡す値のことを **引数（ひきすう）** といいます。

```cpp
#include <iostream>
using namespace std;

void greet(string name) {   // name を受け取る
    cout << "こんにちは、" << name << "さん！" << endl;
}

int main() {
    greet("太郎");
    greet("花子");
    return 0;
}
```

実行結果：
```
こんにちは、太郎さん！
こんにちは、花子さん！
```

引数を複数取ることもできます。

```cpp
void printSum(int a, int b) {
    cout << a << " + " << b << " = " << a + b << endl;
}

printSum(3, 5);   // → 3 + 5 = 8
printSum(10, 20); // → 10 + 20 = 30
```

---

## 戻り値：関数から値を返す

処理の結果を呼び出し元に返すことができます。
返す値のことを **戻り値（もどりち）** といいます。

```cpp
#include <iostream>
using namespace std;

int add(int a, int b) {   // int を返す関数
    return a + b;         // 結果を返す
}

int main() {
    int result = add(3, 5);
    cout << result << endl;   // → 8

    cout << add(10, 20) << endl;   // → 30

    return 0;
}
```

`return` で値を返すと、その値を呼び出し元で受け取ったり、そのまま使ったりできます。

---

## 関数を組み合わせる

```cpp
#include <iostream>
using namespace std;

double calcAverage(int a, int b, int c) {
    int sum = a + b + c;
    return (double)sum / 3;
}

void printAverage(int a, int b, int c) {
    double avg = calcAverage(a, b, c);
    cout << "平均: " << avg << endl;
}

int main() {
    printAverage(80, 70, 90);   // → 平均: 80
    printAverage(60, 55, 75);   // → 平均: 63.3333
    return 0;
}
```

関数が関数を呼ぶことで、処理を小さな部品に分けて整理できます。

---

## 関数を書く位置

関数は `main` より**前**に書くのが基本です。

```cpp
void greet() { ... }   // ← main より前に定義

int main() {
    greet();           // ← ここで呼び出せる
}
```

`main` の後に書きたい場合は、**プロトタイプ宣言**（関数の宣言だけ先に書く）が必要です。

```cpp
void greet();   // プロトタイプ宣言（; で終わる）

int main() {
    greet();
}

void greet() {   // ← 実際の定義は後でOK
    cout << "こんにちは！" << endl;
}
```

---

## 変数のスコープ

関数の中で宣言した変数は、その関数の中でしか使えません。
これを変数の **スコープ（有効範囲）** といいます。

```cpp
void funcA() {
    int x = 10;   // funcA の中だけで有効
}

void funcB() {
    cout << x << endl;   // エラー！x は funcA の中にある
}
```

関数をまたいで値をやりとりするには、引数と戻り値を使いましょう。

---

## まとめ

- 関数とは**処理をまとめて名前をつけたもの**
- `void` は戻り値なし、それ以外は戻り値の型を指定する
- **引数**で関数に値を渡す、**戻り値**で関数から値を受け取る
- 関数は `main` より前に書くか、プロトタイプ宣言を使う
- 関数の中の変数は外から使えない（スコープ）
