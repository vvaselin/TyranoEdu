# 第4章-1: if文の使いかた

---

## 条件によって処理を変えよう

これまでのプログラムは、上から下へ順番に実行されるだけでした。
でも実際のプログラムでは、**状況に応じて違う処理をする**必要があります。

> 「点数が60点以上なら合格、そうでなければ不合格」
> 「入力した数が0より大きければ正の数、そうでなければ…」

こういった「もし〜ならば」という処理を書くのが **`if`文** です。

---

## 基本の形

```cpp
if (条件) {
    // 条件が true のときに実行される
}
```

条件が `true`（成り立つ）のときだけ、`{` `}` の中が実行されます。

```cpp
#include <iostream>
using namespace std;

int main() {
    int score = 80;

    if (score >= 60) {
        cout << "合格です！" << endl;
    }

    return 0;
}
```

実行結果：
```
合格です！
```

`score` が `60` 以上なので、`{}` の中が実行されました。

---

## if-else：どちらかを実行する

条件が成り立たないときに別の処理をしたい場合は `else` を使います。

```cpp
if (条件) {
    // 条件が true のとき
} else {
    // 条件が false のとき
}
```

```cpp
#include <iostream>
using namespace std;

int main() {
    int score = 45;

    if (score >= 60) {
        cout << "合格です！" << endl;
    } else {
        cout << "不合格です…" << endl;
    }

    return 0;
}
```

実行結果：
```
不合格です…
```

`score` が `60` 未満なので、`else` の中が実行されました。

---

## if-else if-else：複数の条件を判定する

条件をさらに細かく分けたいときは `else if` を使います。

```cpp
if (条件1) {
    // 条件1 が true のとき
} else if (条件2) {
    // 条件1 は false、条件2 が true のとき
} else {
    // どの条件も false のとき
}
```

```cpp
#include <iostream>
using namespace std;

int main() {
    int score = 75;

    if (score >= 90) {
        cout << "評価: A" << endl;
    } else if (score >= 70) {
        cout << "評価: B" << endl;
    } else if (score >= 50) {
        cout << "評価: C" << endl;
    } else {
        cout << "評価: D" << endl;
    }

    return 0;
}
```

実行結果：
```
評価: B
```

条件は**上から順番にチェック**されます。
最初に `true` になった条件の `{}` だけが実行され、残りはスキップされます。

---

## 条件のネスト（入れ子）

`if` の中にさらに `if` を書くこともできます。

```cpp
#include <iostream>
using namespace std;

int main() {
    int score = 85;
    bool isStudent = true;

    if (score >= 60) {
        cout << "合格です！" << endl;
        if (isStudent) {
            cout << "学割が適用されます。" << endl;
        }
    } else {
        cout << "不合格です…" << endl;
    }

    return 0;
}
```

実行結果：
```
合格です！
学割が適用されます。
```

入れ子が深くなりすぎると読みにくくなるので、ほどほどにしましょう。

---

## インデント（字下げ）について

`{}` の中の行は、スペースで字下げ（インデント）するのがルールです。

```cpp
// 読みやすい
if (score >= 60) {
    cout << "合格" << endl;
}

// 読みにくい（インデントなし）
if (score >= 60) {
cout << "合格" << endl;
}
```

どちらも動きますが、インデントがないと構造がわかりにくくなります。
**習慣として必ずインデントしましょう。**

> 💡 多くのエディタでは `Tab` キーを押すと自動でインデントされます。

---

## よくあるミス

### ミス1: `=` と `==` を間違える

```cpp
if (score = 60) {   // ← = は代入！比較ではない
```

条件の中で「等しいか比較する」には `==` を使います（次のページで詳しく説明します）。

### ミス2: `{}` を忘れる

```cpp
if (score >= 60)
    cout << "合格" << endl;
    cout << "おめでとう" << endl;   // ← これは if の外！常に実行される
```

`{}` がないと直後の1行しか `if` に入りません。
**複数行を `if` に入れるときは必ず `{}` で囲みましょう。**
慣れないうちは1行でも `{}` をつける習慣をおすすめします。

---

## まとめ

- `if` で「もし〜ならば」という処理を書ける
- `else` で条件が成り立たないときの処理を書ける
- `else if` で複数の条件を順番にチェックできる
- 条件は**上から順番**に評価され、最初に `true` になったものだけ実行される
- `{}` の中は必ずインデントする
