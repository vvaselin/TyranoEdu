## 条件を詳しく書こう

前のページで `if (score >= 60)` という条件を書きました。
この `>=` のような、2つの値を比べる記号を **比較演算子** といいます。

また、複数の条件を組み合わせる **論理演算子** も一緒に紹介します。

---

## 比較演算子の一覧

| 演算子 | 意味 | 例 | 結果 |
|--------|------|----|------|
| `==` | 等しい | `3 == 3` | `true` |
| `!=` | 等しくない | `3 != 4` | `true` |
| `<` | より小さい | `3 < 5` | `true` |
| `>` | より大きい | `5 > 3` | `true` |
| `<=` | 以下 | `3 <= 3` | `true` |
| `>=` | 以上 | `4 >= 3` | `true` |

比較の結果は `true`（成り立つ）か `false`（成り立たない）のどちらかになります。

---

## よく混乱する「等しい」の比較

数学では `=` が「等しい」ですが、C++では `=` は**代入**です。
比較には `==`（イコール2つ）を使います。

```cpp
int x = 5;       // x に 5 を代入する
if (x == 5) {    // x が 5 と等しいか比較する
    cout << "xは5です" << endl;
}
```

この違いはよく間違えるので、しっかり覚えておきましょう。

```cpp
if (x = 5) {   // ← バグ！比較ではなく代入になってしまう
```

---

## 比較演算子を使った例

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10;
    int b = 20;

    if (a == b) {
        cout << "aとbは等しい" << endl;
    } else {
        cout << "aとbは等しくない" << endl;
    }

    if (a < b) {
        cout << "aはbより小さい" << endl;
    }

    if (a != b) {
        cout << "aとbは違う値" << endl;
    }

    return 0;
}
```

実行結果：
```
aとbは等しくない
aはbより小さい
aとbは違う値
```

## 論理演算子：条件を組み合わせる

複数の条件を組み合わせたいときは **論理演算子** を使います。

| 演算子 | 読み方 | 意味 | 例 |
|--------|--------|------|----|
| `&&` | かつ（AND） | 両方 `true` なら `true` | `a > 0 && a < 10` |
| `\|\|` | または（OR） | どちらか `true` なら `true` | `a == 0 \|\| a == 1` |
| `!` | でない（NOT） | `true` と `false` を反転 | `!isStudent` |

---

### `&&`（かつ）

**両方の条件が成り立つ**ときに `true` になります。

```cpp
int score = 75;
bool submitted = true;

if (score >= 60 && submitted) {
    cout << "合格！（提出済み・60点以上）" << endl;
}
```

`score >= 60` かつ `submitted` が `true` のとき合格。
どちらか一方でも満たさなければ合格になりません。

---

### `||`（または）

**どちらか一方でも成り立つ**ときに `true` になります。

```cpp
int month = 8;

if (month == 7 || month == 8) {
    cout << "夏休みです！" << endl;
}
```

7月または8月なら夏休みと判定します。

---

### `!`（でない）

条件の `true` / `false` を**反転**させます。

```cpp
bool isRaining = false;

if (!isRaining) {
    cout << "今日は晴れています" << endl;
}
```

`isRaining` が `false` なので、`!isRaining` は `true` になります。

---

## 組み合わせて使う

```cpp
#include <iostream>
using namespace std;

int main() {
    int age = 25;
    bool hasLicense = true;

    if (age >= 18 && hasLicense) {
        cout << "車を運転できます" << endl;
    } else if (age >= 18 && !hasLicense) {
        cout << "免許を取れば運転できます" << endl;
    } else {
        cout << "年齢が足りません" << endl;
    }

    return 0;
}
```

実行結果：
```
車を運転できます
```

---

## 真偽値（bool）との比較

`bool` 型の変数は、そのまま条件として使えます。

```cpp
bool isStudent = true;

// この2つは同じ意味
if (isStudent == true) { ... }
if (isStudent) { ... }           // ← こちらの方がスッキリ

// この2つも同じ意味
if (isStudent == false) { ... }
if (!isStudent) { ... }          // ← こちらの方がスッキリ
```

`== true` は省略して書くのが一般的です。

---

## まとめ

- 比較演算子: `==` `!=` `<` `>` `<=` `>=`
- `=`（代入）と `==`（比較）を混同しないように注意
- `&&`（かつ）で複数条件をすべて満たすか判定できる
- `||`（または）でどれか一つを満たすか判定できる
- `!`（でない）で条件を反転できる
