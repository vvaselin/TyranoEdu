## `<>` って何だろう？

これまで `vector<int>` や `vector<string>` という書き方をしてきました。
この `<int>` や `<string>` の部分が **テンプレート** の仕組みです。

```cpp
vector<int>     // int を入れる vector
vector<string>  // string を入れる vector
vector<double>  // double を入れる vector
```

「型だけ違って、やることは同じ」という処理を、1つの定義でまかなえるのがテンプレートです。

## テンプレート関数を作ってみよう

たとえば、2つの値のうち大きい方を返す関数を考えます。

テンプレートなしだと、型ごとに関数を書かなければなりません。

```cpp
int    maxInt(int a, int b)       { return a > b ? a : b; }
double maxDouble(double a, double b) { return a > b ? a : b; }
// string 用も必要？...
```

テンプレートを使うと、1つにまとめられます。

```cpp
template <typename T>
T myMax(T a, T b) {
    return a > b ? a : b;
}
```

---

## テンプレート関数の使い方

```cpp
#include <iostream>
using namespace std;

template <typename T>
T myMax(T a, T b) {
    return a > b ? a : b;
}

int main() {
    cout << myMax(3, 7) << endl;          // → 7（int として使う）
    cout << myMax(1.5, 0.8) << endl;      // → 1.5（double として使う）
    cout << myMax('a', 'z') << endl;      // → z（char として使う）
    return 0;
}
```

呼び出すと、コンパイラが引数の型を見て自動的に `T` に当てはめてくれます。
明示的に型を指定することもできます。

```cpp
myMax<int>(3, 7);      // T = int として使う
myMax<double>(3, 7);   // T = double として使う
```

## `typename` と `class`

テンプレートの宣言には `typename` の代わりに `class` と書くこともできます。
どちらも意味は同じです。

```cpp
template <typename T>  // こちらが一般的
template <class T>     // こちらでも動く
```

---

## 複数の型パラメータ

型パラメータは複数持てます。

```cpp
template <typename T1, typename T2>
void printPair(T1 a, T2 b) {
    cout << a << " : " << b << endl;
}

printPair(1, "one");        // → 1 : one
printPair("score", 100);    // → score : 100
```

## テンプレートクラス

クラスにもテンプレートを使えます。
`vector` 自体がテンプレートクラスとして定義されています。

```cpp
template <typename T>
class Box {
public:
    T value;

    Box(T v) {
        value = v;
    }

    void show() {
        cout << value << endl;
    }
};

Box<int> intBox(42);
intBox.show();            // → 42

Box<string> strBox("Hello");
strBox.show();            // → Hello
```

---

## テンプレートのまとめ

- テンプレートは「型だけ違う同じ処理」を1つにまとめる仕組み
- `template <typename T>` で宣言し、`T` の部分に実際の型が当てはまる
- `vector<int>` などの `<>` はテンプレートの型指定
- 関数にもクラスにも使える
