## 文字列の基本操作（おさらい）

| 操作 | 書き方 | 例 |
|------|--------|----|
| 長さ | `.size()` / `.length()` | `s.size()` |
| 連結 | `+` / `+=` | `s1 + s2` |
| 1文字取得 | `[i]` | `s[0]` |
| 部分文字列 | `.substr(開始, 文字数)` | `s.substr(2, 3)` |
| 検索 | `.find(文字列)` | `s.find("ab")` |
| 比較 | `==` / `!=` / `<` | `s1 == s2` |

## 文字列の変換

### 数値 → string

```cpp
int n = 42;
double d = 3.14;

string s1 = to_string(n);   // → "42"
string s2 = to_string(d);   // → "3.140000"
```

### string → 数値

```cpp
string s = "123";

int n    = stoi(s);    // → 123（int）
long l   = stol(s);    // → 123（long）
double d = stod(s);    // → 123.0（double）
```

| 関数 | 変換先 | 意味 |
|------|--------|------|
| `stoi()` | `int` | string to int |
| `stol()` | `long` | string to long |
| `stod()` | `double` | string to double |

## 文字列の置換（replace）

`.replace(開始位置, 文字数, 新しい文字列)` で文字列の一部を置き換えます。

```cpp
string s = "Hello, World!";
s.replace(7, 5, "C++");
cout << s << endl;   // → Hello, C++!
```

---

## 文字列の挿入（insert）と削除（erase）

```cpp
string s = "Hello!";

// 挿入：5番目の位置に ", World" を挿入
s.insert(5, ", World");
cout << s << endl;   // → Hello, World!

// 削除：5文字目から7文字削除
s.erase(5, 7);
cout << s << endl;   // → Hello!
```

## スペース区切りで分割する

C++には文字列分割の専用関数がないため、`stringstream` を使うのが定番です。

```cpp
#include <iostream>
#include <string>
#include <sstream>
#include <vector>
using namespace std;

int main() {
    string line = "apple banana cherry";
    stringstream ss(line);
    string word;
    vector<string> words;

    while (ss >> word) {
        words.push_back(word);
    }

    for (const string& w : words) {
        cout << w << endl;
    }

    return 0;
}
```

実行結果：
```
apple
banana
cherry
```

## 大文字・小文字の変換

`<cctype>` を使うと1文字ずつ変換できます。

```cpp
#include <cctype>

char c = 'a';
cout << (char)toupper(c) << endl;   // → A
cout << (char)tolower('B') << endl; // → b
```

文字列全体を変換するには、1文字ずつ処理します。

```cpp
string s = "Hello";
for (char& c : s) {
    c = toupper(c);
}
cout << s << endl;   // → HELLO
```

## 文字列の先頭・末尾の空白を取り除く（trim）

C++には trim 関数がないため、自分で書きます。

```cpp
#include <algorithm>

string trim(const string& s) {
    int start = s.find_first_not_of(" \t\n");
    int end   = s.find_last_not_of(" \t\n");
    if (start == string::npos) return "";
    return s.substr(start, end - start + 1);
}

string s = "  Hello, World!  ";
cout << trim(s) << endl;   // → Hello, World!
```

## string と char の関係

`string` は `char` の集まりです。`[i]` で取り出した要素は `char` 型です。

```cpp
string s = "Hello";
char c = s[0];          // 'H'（char型）
cout << c << endl;      // → H

// char と string の変換
string fromChar = string(1, c);   // char → string
```

---

## まとめ

- `stoi()` / `stod()` で文字列を数値に変換、`to_string()` で逆変換
- `replace()` / `insert()` / `erase()` で文字列を編集できる
- `stringstream` でスペース区切りの分割ができる
- 大文字・小文字の変換は `toupper()` / `tolower()` を使う
- `string` の各文字は `char` 型
