## 変数に値を入れる（代入）

変数に値を入れることを **代入** といいます。
`=` は「右側の値を左側の変数に入れる」という意味です。

```cpp
int score;    // 宣言
score = 80;   // 代入
```

数学の `=`（等しい）とは意味が違うので注意してください。

```cpp
int x = 5;
x = x + 1;   // 「x に x+1 を代入する」→ x は 6 になる
```

数学では `x = x + 1` はありえませんが、プログラムでは「今の `x` に1を足した値を、また `x` に入れ直す」という意味になります。

---

## 変数の中身を表示する

`cout` に変数をそのまま渡すと、中身が表示されます。

```cpp
#include <iostream>
using namespace std;

int main() {
    int score = 85;
    cout << score << endl;        // → 85
    cout << "点数: " << score << "点" << endl;   // → 点数: 85点
    return 0;
}
```

文字列と変数は `<<` でつなげて一緒に表示できます。

---

## 変数の値を変える

変数の値は、後から何度でも上書きできます。

```cpp
#include <iostream>
using namespace std;

int main() {
    int score = 50;
    cout << "最初の点数: " << score << endl;   // → 50

    score = 80;
    cout << "更新後の点数: " << score << endl; // → 80

    score = 100;
    cout << "最終的な点数: " << score << endl; // → 100

    return 0;
}
```

実行結果：
```
最初の点数: 50
更新後の点数: 80
最終的な点数: 100
```

---

## 変数同士でやりとりする

変数の値を別の変数にコピーすることもできます。

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10;
    int b = a;    // a の中身を b にコピー

    cout << "a = " << a << endl;   // → 10
    cout << "b = " << b << endl;   // → 10

    a = 99;   // a だけ変えても...

    cout << "a = " << a << endl;   // → 99
    cout << "b = " << b << endl;   // → 10（b はそのまま）

    return 0;
}
```

> 💡 `b = a` は「`a` と `b` をつなぐ」のではなく「**コピーする**」だけです。
> その後 `a` を変えても `b` には影響しません。

## 変数名のルール

変数につける名前には、いくつかのルールがあります。

**使えるもの：**
- 英字（a〜z、A〜Z）
- 数字（0〜9）※ 先頭には使えない
- アンダースコア（`_`）

**使えないもの：**
- 数字から始まる名前（`1score` はNG）
- スペース（`my score` はNG）
- C++の予約語（`int`、`return`、`if` など）

```cpp
int score = 90;        // OK
int my_score = 90;     // OK（アンダースコアOK）
int myScore = 90;      // OK（キャメルケースとも呼ぶ）
int 1score = 90;       // NG（数字から始まっている）
int my score = 90;     // NG（スペースが入っている）
int int = 90;          // NG（予約語）
```

**大文字・小文字は別物として扱われます：**

```cpp
int score = 10;
int Score = 20;   // score とは別の変数
```

---

## 意味のある名前をつけよう

変数名は `a` や `x` でも動きますが、**何を表しているかわかる名前**にすると、
あとでコードを読み返したときに理解しやすくなります。

```cpp
// 悪い例：名前から意味がわからない
int a = 18;
double b = 170.5;

// 良い例：名前から意味が伝わる
int age = 18;
double height = 170.5;
```

> 💡 プログラムは書く時間より**読む時間の方が長い**といわれます。
> 未来の自分や他の人が読んでもわかる名前をつける習慣をつけましょう。

---

## まとめ

- `=` は「右の値を左の変数に代入する」という意味
- 変数の値は後から何度でも上書きできる
- 変数同士のコピーは「つなぐ」のではなく「値を複製する」
- 変数名は**意味のわかる名前**にすると読みやすいコードになる
