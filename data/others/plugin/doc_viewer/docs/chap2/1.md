## 第2章：変数

プログラミングでは、数値や文字を一時的に保存しておくための「箱」が必要です。これを **変数（へんすう）** と呼びます。

### 2.1 変数の宣言と型

変数を使うには、「どんな種類のデータを入れるか」を指定する必要があります。

| 型の種類 | 扱えるデータ | 例 |
| --- | --- | --- |
| **int** | 整数 | 10, -5, 0 |
| **double** | 小数（実数） | 3.14, -0.5 |
| **char** | 1文字 | 'A', 'z' |
| **string** | 文字列 | "Hello", "C++" |
| **bool** | 真偽値（正しいか否か） | true, false |
