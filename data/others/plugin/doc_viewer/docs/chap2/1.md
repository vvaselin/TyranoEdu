# 第2章-1: 変数の宣言と型

[← 前: 文字の入力](chap1/2.md)　／　[→ 次: 変数の使い方](chap2/2.md)

---

## 変数って何だろう？

プログラムでは、計算の結果やユーザーが入力した値を、一時的に覚えておく必要があります。
その「覚えておく場所」のことを **変数** といいます。

変数は **ラベルの貼ってある箱** のようなものです。

```
┌─────────┐
│   42    │  ← 中に値が入っている
└─────────┘
    age       ← 箱の名前（変数名）
```

この箱に名前をつけておけば、あとから「`age` の箱の中身を教えて」と取り出せます。

---

## 変数を使うための2ステップ

変数を使うには、まず **宣言** が必要です。
「この名前の箱を用意します」とコンピュータに伝える作業です。

```cpp
int age;      // ステップ1: 宣言（箱を用意する）
age = 20;     // ステップ2: 代入（箱に値を入れる）
```

宣言と代入を同時に行うこともできます。

```cpp
int age = 20;   // 宣言と同時に値を入れる
```

---

## 型って何だろう？

C++の箱には **型（かた）** があります。
「整数しか入れられない箱」「小数も入れられる箱」のように、
**入れられる値の種類があらかじめ決まっている**のです。

なぜ型が必要かというと、コンピュータは値の種類によって内部での扱い方が変わるからです。
型を指定することで、コンピュータが正しく効率よく処理できます。

---

## 主な型の種類

### `int`（インテジャー）— 整数

```cpp
int score = 100;
int year = 2024;
int temperature = -5;   // 負の数もOK
```

小数点のない整数を扱います。日常的に一番よく使う型です。

---

### `double`（ダブル）— 小数

```cpp
double height = 170.5;
double pi = 3.14159;
double weight = -0.5;   // 負の小数もOK
```

小数点のある数を扱います。身長・体重・割合などに使います。

> 💡 小数を扱う型には `float` もありますが、精度が高い `double` を使う方が一般的です。

---

### `char`（キャラ）— 1文字

```cpp
char grade = 'A';
char symbol = '!';
char digit = '3';   // 数字も1文字として扱える
```

文字1つだけを扱います。値は **シングルクォーテーション `'`** で囲みます。

> ⚠️ `char` に入れられるのは**1文字だけ**です。`'AB'` はエラーになります。

---

### `string`（ストリング）— 文字列

```cpp
string name = "田中太郎";
string message = "Hello, World!";
string empty = "";   // 空の文字列もOK
```

複数の文字からなる「文字列」を扱います。値は **ダブルクォーテーション `"`** で囲みます。

`string` を使うには、先頭に `#include <string>` が必要です。
（`#include <iostream>` と一緒に書いておきましょう）

---

### `bool`（ブール）— 真偽値

```cpp
bool isStudent = true;
bool isRaining = false;
```

`true`（真・はい）か `false`（偽・いいえ）の2つの値だけを扱います。
条件分岐（第4章）でよく登場します。

---

## 型のまとめ

| 型 | 読み方 | 扱う値 | 例 |
|----|--------|--------|-----|
| `int` | インテジャー | 整数 | `42`, `-10`, `0` |
| `double` | ダブル | 小数 | `3.14`, `-0.5` |
| `char` | キャラ | 1文字 | `'A'`, `'!'` |
| `string` | ストリング | 文字列 | `"Hello"` |
| `bool` | ブール | 真偽値 | `true`, `false` |

---

## サンプルコード

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string name = "太郎";
    int age = 18;
    double height = 170.5;

    cout << "名前: " << name << endl;
    cout << "年齢: " << age << "歳" << endl;
    cout << "身長: " << height << "cm" << endl;

    return 0;
}
```

実行結果：
```
名前: 太郎
年齢: 18歳
身長: 170.5cm
```

---

## よくあるミス

### ミス1: 型を書き忘れる

```cpp
age = 20;   // ← 型がない！
```

変数を初めて使うときは、必ず型を書いて宣言してください。

### ミス2: `char` に複数文字を入れる

```cpp
char name = "太郎";   // ← char は1文字だけ！string を使おう
```

### ミス3: `string` と `char` のクォーテーションを間違える

```cpp
string message = 'Hello';   // ← string はダブルクォーテーション！
char grade = "A";           // ← char はシングルクォーテーション！
```

---

## まとめ

- 変数とは、値を入れておく「名前のついた箱」
- 変数を使う前に **型と名前を宣言** する
- 主な型: `int`（整数）、`double`（小数）、`char`（1文字）、`string`（文字列）、`bool`（真偽値）
