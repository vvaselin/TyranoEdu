## 計算をしてみよう

C++では、変数や数値を使って計算ができます。
足し算・引き算・掛け算・割り算の4つを **四則演算** といいます。

---

## 算術演算子

計算に使う記号を **演算子** といいます。
数学の記号と少し違うものがあるので確認しておきましょう。

| 演算子 | 意味 | 例 | 結果 |
|--------|------|----|------|
| `+` | 足し算 | `3 + 2` | `5` |
| `-` | 引き算 | `3 - 2` | `1` |
| `*` | 掛け算 | `3 * 2` | `6` |
| `/` | 割り算 | `6 / 2` | `3` |
| `%` | 余り | `7 % 3` | `1` |

掛け算は `×` ではなく `*`、割り算は `÷` ではなく `/` を使います。

---

## 基本的な使い方

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10;
    int b = 3;

    cout << a + b << endl;   // → 13
    cout << a - b << endl;   // → 7
    cout << a * b << endl;   // → 30
    cout << a / b << endl;   // → 3
    cout << a % b << endl;   // → 1

    return 0;
}
```

計算結果を変数に代入することもできます。

```cpp
int sum = a + b;
cout << "合計: " << sum << endl;   // → 合計: 13
```

---

## 余り（`%`）について

`%`（パーセント）は聞き慣れないかもしれませんが、とても便利な演算子です。
割り算の **余り** を求めます。

```
7 ÷ 3 = 2 あまり 1
         ↑         ↑
        商(몫)      余り → 7 % 3 の結果
```

**よく使う場面：**
- 偶数・奇数の判定（2で割った余りが0なら偶数）
- 3の倍数かどうかの判定（3で割った余りが0なら倍数）

```cpp
int n = 7;
cout << n % 2 << endl;   // → 1（奇数）

int m = 6;
cout << m % 2 << endl;   // → 0（偶数）
```

---

## 整数同士の割り算に注意！

`int` 同士で割り算をすると、小数点以下は**切り捨て**られます。

```cpp
int a = 7;
int b = 2;
cout << a / b << endl;   // → 3（3.5 ではない！）
```

小数点以下も含めた結果が欲しいときは、`double` を使いましょう。

```cpp
double a = 7.0;
double b = 2.0;
cout << a / b << endl;   // → 3.5
```

または、片方だけ `double` にしても小数になります。

```cpp
int a = 7;
int b = 2;
cout << (double)a / b << endl;   // → 3.5
```

`(double)` は「この値を `double` として扱う」という意味です（キャストといいます）。

## 複合代入演算子

「変数に計算結果を上書きする」操作は、よく登場します。

```cpp
int x = 10;
x = x + 5;   // x に 5 を足して上書き
```

これを短く書ける **複合代入演算子** があります。

| 長い書き方 | 短い書き方 | 意味 |
|------------|------------|------|
| `x = x + 5` | `x += 5` | 5を足す |
| `x = x - 5` | `x -= 5` | 5を引く |
| `x = x * 5` | `x *= 5` | 5を掛ける |
| `x = x / 5` | `x /= 5` | 5で割る |
| `x = x % 5` | `x %= 5` | 5で割った余り |

```cpp
int score = 50;
score += 20;   // score = 50 + 20 = 70
score -= 10;   // score = 70 - 10 = 60
cout << score << endl;   // → 60
```

---

## インクリメント・デクリメント

「1だけ増やす」「1だけ減らす」操作はとてもよく使うので、専用の演算子があります。

| 演算子 | 意味 | 同じ意味 |
|--------|------|----------|
| `x++` | 1増やす | `x = x + 1` |
| `x--` | 1減らす | `x = x - 1` |

```cpp
int count = 0;
count++;   // → 1
count++;   // → 2
count++;   // → 3
cout << count << endl;   // → 3
```

`++` はインクリメント、`--` はデクリメントといいます。
繰り返し処理（第5章）でとくによく登場します。

---

## 計算の優先順位

数学と同じように、掛け算・割り算は足し算・引き算より先に計算されます。
また、`()` を使うと優先順位を変えられます。

```cpp
cout << 2 + 3 * 4 << endl;     // → 14（3*4=12 を先に計算）
cout << (2 + 3) * 4 << endl;   // → 20（2+3=5 を先に計算）
```

---

## サンプル: 平均点を計算する

```cpp
#include <iostream>
using namespace std;

int main() {
    int english = 80;
    int math = 70;
    int science = 90;

    double average = (english + math + science) / 3.0;

    cout << "英語: " << english << "点" << endl;
    cout << "数学: " << math << "点" << endl;
    cout << "理科: " << science << "点" << endl;
    cout << "平均: " << average << "点" << endl;

    return 0;
}
```

実行結果：
```
英語: 80点
数学: 70点
理科: 90点
平均: 80点
```

> 💡 `3` ではなく `3.0` で割っているのは、小数の結果を得るためです。

---

## まとめ

- 四則演算は `+` `-` `*` `/`、余りは `%`
- `int` 同士の割り算は小数が切り捨てられる
- 複合代入演算子（`+=` など）を使うと短く書ける
- `x++` / `x--` で1だけ増減できる
- `()` で計算の優先順位を変えられる
