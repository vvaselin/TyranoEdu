## 第1章：入出力

まずは、プログラムと対話するための基本である「画面への表示（出力）」と「キーボードからの読み取り（入力）」を学びます。

### 1.1 文字の出力

C++で文字を表示するには `std::cout` を使います。

```cpp
#include <iostream> // 入出力のための準備

int main() {
    std::cout << "こんにちは、世界！" << std::endl;
    return 0;
}

```

* `std::cout`: 画面に出力せよ、という命令。
* `<<`: データを送る向きを表します。
* `std::endl`: 改行（End of Line）を意味します。
