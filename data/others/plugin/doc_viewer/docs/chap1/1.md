## 画面に文字を表示させよう

プログラムの基本中の基本は、**画面に文字を表示すること**です。
まずはここから始めましょう。

---

## サンプルコードを見てみよう

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}
```

実行結果：
```
Hello, World!
```

たった5行ですが、いくつかのパーツが組み合わさっています。
一つずつ見ていきましょう。

---

## コードの解説

### `#include <iostream>`

```cpp
#include <iostream>
```

`iostream`（アイ・オー・ストリーム）は、文字の入出力に必要な機能をまとめたファイルです。
`#include` は「このファイルを使います」という宣言です。

画面に文字を表示したいときは、**必ずこの1行が必要**です。

> 💡 `iostream` の `io` は input/output（入力・出力）の略、`stream` は「流れ」という意味です。

---

### `using namespace std;`

```cpp
using namespace std;
```

この行は「`std`という名前のグループの機能を省略して使えるようにする」という意味です。

これがないと、`cout` の代わりに `std::cout` と書かなければならず、少し面倒になります。
今は「おまじない」として覚えておけば大丈夫です。

> 📖 **もっと知りたい方へ:** [コラム: `std::` って何？](chap1/column1.md)

---

### `int main() { ... }`

```cpp
int main() {
    // ここにプログラムの内容を書く
    return 0;
}
```

C++のプログラムは必ず **`main`関数** からスタートします。
`{` と `}` の間に書いた内容が、上から順番に実行されます。

`return 0;` は「プログラムが正常に終わりました」という合図です。

---

### `cout << "..." << endl;`

```cpp
cout << "Hello, World!" << endl;
```

これが実際に文字を表示している部分です。

| パーツ | 読み方 | 意味 |
|--------|--------|------|
| `cout` | シー・アウト | 画面への出力 |
| `<<` | 矢印 | 右のものを左に送る |
| `"Hello, World!"` | — | 表示したい文字（**ダブルクォーテーションで囲む**） |
| `endl` | エンド・エル | 改行（次の行へ移る） |
| `;` | セミコロン | 文の終わり |

`<<` を「流し込む」イメージで覚えると分かりやすいです。
「`"Hello, World!"`と`endl`を、`cout`（画面）に流し込む」という感じです。

## いろいろ試してみよう

### 好きな文字を表示する

`"` と `"` の間の文字を変えると、好きな文字を表示できます。

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "こんにちは！" << endl;
    return 0;
}
```

実行結果：
```
こんにちは！
```

---

### 複数の行を表示する

`cout << ... << endl;` を何行も書けば、複数行表示できます。

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "1行目" << endl;
    cout << "2行目" << endl;
    cout << "3行目" << endl;
    return 0;
}
```

実行結果：
```
1行目
2行目
3行目
```

> 💡 プログラムは **上から順番に** 実行されます。

---

### `endl` を省略するとどうなる？

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "こんにちは、";
    cout << "世界！" << endl;
    return 0;
}
```

実行結果：
```
こんにちは、世界！
```

`endl` がないと改行されないので、次の `cout` の内容がつながって表示されます。

---

## よくあるミス

### ミス1: セミコロンを忘れる

```cpp
cout << "Hello" << endl   // ← ; がない！
```

C++では、1つの文の終わりには必ず `;` が必要です。忘れるとエラーになります。

### ミス2: ダブルクォーテーションを忘れる・間違える

```cpp
cout << Hello << endl;   // ← " " がない！
```

表示したい文字は必ず `"` で囲みましょう。

### ミス3: 大文字・小文字を間違える

```cpp
Cout << "Hello" << endl;   // ← 大文字になっている！
```

C++は大文字・小文字を区別します。`cout` は全部小文字です。

---

## まとめ

- 画面に文字を表示するには `cout << "文字" << endl;` を使う
- `#include <iostream>` と `using namespace std;` は毎回必要
- プログラムは `int main()` の中に書く
- 文の終わりには `;` を忘れずに
