## ユーザーからの入力を受け取ろう

前のページでは、プログラムが決まった文字を表示する方法を学びました。
今度は、**ユーザーがキーボードで入力した内容**をプログラムで受け取ってみましょう。

入力を受け取れるようになると、プログラムがぐっと「生きている」感じになります！

---

## サンプルコードを見てみよう

```cpp
#include <iostream>
using namespace std;

int main() {
    int number;
    cout << "数字を入力してください: ";
    cin >> number;
    cout << "入力された数字は " << number << " です。" << endl;
    return 0;
}
```

実行例：
```
数字を入力してください: 42
入力された数字は 42 です。
```

ユーザーが `42` と入力すると、それを受け取って表示しています。

---

## コードの解説

### `int number;`

```cpp
int number;
```

入力された値を保存するための **「箱」** を用意しています。
この「箱」のことを **変数** といいます。

`int` は「整数を入れる箱」という意味です。
`number` はその箱につけた名前です。

> 💡 変数については、次の第2章でくわしく学びます。今は「入力を受け取るための箱を用意した」と覚えておきましょう。

---

### `cin >> number;`

```cpp
cin >> number;
```

`cin`（シー・イン）が、キーボードからの入力を受け取る命令です。

`cout` と `cin` はセットで覚えましょう：

| 命令 | 読み方 | 方向 | 使い方 |
|------|--------|------|--------|
| `cout` | シー・アウト | プログラム → 画面 | `cout << 表示したい内容` |
| `cin` | シー・イン | キーボード → プログラム | `cin >> 受け取る変数` |

`<<` と `>>` の向きに注目してください。**データが流れていく方向**を表しています。

---

### `cout << "..." << number << "..." << endl;`

```cpp
cout << "入力された数字は " << number << " です。" << endl;
```

`<<` はいくつでも続けて書けます。
文字列と変数を組み合わせて表示することができます。

---

## 文字列を入力する

数字だけでなく、文字も入力できます。

```cpp
#include <iostream>
using namespace std;

int main() {
    string name;
    cout << "あなたの名前を入力してください: ";
    cin >> name;
    cout << "こんにちは、" << name << "さん！" << endl;
    return 0;
}
```

実行例：
```
あなたの名前を入力してください: 太郎
こんにちは、太郎さん！
```

文字列を受け取るときは、`int` の代わりに `string` を使います。

> ⚠️ `cin >>` でスペースを含む文字列を受け取る場合は少し工夫が必要です。「田中 太郎」のように空白が入る場合は`getline`という方法を使います（今は気にしなくて大丈夫です）。

---

## 複数の入力を受け取る

```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cout << "1つ目の数字: ";
    cin >> a;
    cout << "2つ目の数字: ";
    cin >> b;
    cout << a << " と " << b << " を入力しました。" << endl;
    return 0;
}
```

実行例：
```
1つ目の数字: 10
2つ目の数字: 20
10 と 20 を入力しました。
```

このように、`cin` を複数回使えば複数の入力を受け取れます。

---

## よくあるミス

### ミス1: 矢印の向きを間違える

```cpp
cin << number;   // ← << になっている！
```

`cin` は `>>` を使います。`cout` とは逆向きです。

### ミス2: 変数を用意し忘れる

```cpp
cin >> number;   // ← number を宣言していない！
```

`cin` で受け取る前に、必ず変数を宣言（準備）しておく必要があります。

---

## まとめ

- キーボードからの入力を受け取るには `cin >> 変数名;` を使う
- `cout`（出力）と `cin`（入力）はセットで覚えよう
- `int` は整数、`string` は文字列を入れる変数の型
- `<<` と `>>` の向きはデータが流れる向き
