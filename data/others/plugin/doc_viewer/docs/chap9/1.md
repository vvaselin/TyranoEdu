# 第9章-1: 参照（&）

## 変数に別名をつける

**参照** とは、ある変数に対して**別名（エイリアス）をつける**仕組みです。

```cpp
int a = 10;
int& b = a;   // b は a の別名
```

`b` は `a` とは別の変数ではなく、**同じ場所を指す別名**です。
どちらを変えても、もう一方も変わります。

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10;
    int& b = a;   // b は a の参照

    cout << a << endl;   // → 10
    cout << b << endl;   // → 10

    b = 99;   // b を変えると…
    cout << a << endl;   // → 99（a も変わる！）

    return 0;
}
```

---

## 参照はなぜ必要？

参照が本当に役立つのは、**関数に値を渡すとき**です。

通常、関数に変数を渡すと**コピーが渡される**ので、関数の中で変えても元の変数は変わりません。

```cpp
#include <iostream>
using namespace std;

void addOne(int x) {
    x++;   // コピーを変えているだけ
}

int main() {
    int n = 10;
    addOne(n);
    cout << n << endl;   // → 10（変わっていない）
    return 0;
}
```

参照を使うと、元の変数そのものを渡せます。

```cpp
#include <iostream>
using namespace std;

void addOne(int& x) {   // 参照で受け取る
    x++;   // 元の変数を直接変える
}

int main() {
    int n = 10;
    addOne(n);
    cout << n << endl;   // → 11（変わった！）
    return 0;
}
```

## 参照渡しの使いどころ

### 関数で複数の値を返したい

関数の戻り値は1つだけですが、参照を使えば複数の変数を同時に変更できます。

```cpp
#include <iostream>
using namespace std;

void minmax(int a, int b, int& minVal, int& maxVal) {
    if (a < b) {
        minVal = a;
        maxVal = b;
    } else {
        minVal = b;
        maxVal = a;
    }
}

int main() {
    int lo, hi;
    minmax(7, 3, lo, hi);
    cout << "最小: " << lo << endl;   // → 3
    cout << "最大: " << hi << endl;   // → 7
    return 0;
}
```

---

### 大きなデータのコピーを避ける

`string` や `vector` など大きなデータを関数に渡すとき、コピーするとメモリと時間が余分にかかります。
参照渡しにすると、コピーせずに渡せます。

```cpp
// コピーが発生する（データが大きいほど遅い）
void print(string s) {
    cout << s << endl;
}

// 参照渡し（コピーなし・速い）
void print(const string& s) {
    cout << s << endl;
}
```

`const` をつけると「読むだけで変更しない」という意味になり、安全に参照渡しできます。

---

## const参照

変更させたくない参照には `const` をつけます。

```cpp
void show(const int& x) {
    cout << x << endl;
    x = 99;   // エラー！const なので変更できない
}
```

大きなデータを関数に渡すときは **`const 型&`** を使うのがC++の定番スタイルです。

---

## まとめ

- 参照（`&`）は変数に**別名をつける**仕組み
- 関数の引数に `&` をつけると、コピーではなく**元の変数を直接操作**できる
- 大きなデータを渡すときは `const 型&` でコピーを避けられる
- 参照は宣言と同時に初期化しなければならない（後から別の変数に変更不可）
