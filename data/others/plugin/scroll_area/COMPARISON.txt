■ 横スクロール向きの用途
  ✓ クエスト選択画面（カードを横に並べる）
  ✓ キャラクター一覧（キャラを横に並べる）
  ✓ アイテムショップ（商品を横に並べる）
  ✓ タイムライン表示（時系列を左から右へ）
  ✓ ギャラリー表示（画像を横に並べる）

■ 縦スクロール向きの用途
  ✓ メニューリスト（項目を縦に並べる）
  ✓ チャット履歴（会話を上から下へ）
  ✓ タスクリスト（TODO項目を縦に並べる）
  ✓ ニュースフィード（記事を縦に並べる）
  ✓ 設定画面（設定項目を縦に並べる）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

どちらを使うべきか迷ったら：

✓ 横スクロールを選ぶべき場合
  → 画像やカードなど、視覚的に大きな要素を並べたい
  → 進行や時間の流れを左→右で表現したい
  → スマホで横持ちでの使用を想定している

✓ 縦スクロールを選ぶべき場合
  → テキストやリスト項目など、縦に並べたほうが読みやすい
  → 従来のWebページのような操作感にしたい
  → スマホで縦持ちでの使用を想定している

✓ 両方使う場合
  → メインメニュー（縦）とサブメニュー（横）など、
    階層構造を表現したい場合

  ※ただし、同時に表示すると操作が複雑になるので注意
